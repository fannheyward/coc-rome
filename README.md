# coc-rome

[Rome](https://rome.tools) extension for coc.nvim

## Install

`:CocInstall coc-rome`

## Configurations

- `rome.enable`: enable coc-rome, default `true`
- `rome.bin`: custom rome binary path. By default, coc-rome only uses local rome binary in your project.
